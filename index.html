이미지

생성

위성 이미지 생성을 요청합니다. 동적으로 생성되는 지도를 크롤링 해와야 하기 때문에 별도의 서버에 작업을 요청하고, 크롤링이 완료되면 자신의 이미지를 요청할 수 있는 키 값이 발급됩니다.

request

Parameter Description

companyType

지도를 제공하는 회사

lat

위도

lng

경도

layerMode

도시계획 레이어 적용 유무

level

지도 캡쳐 반경. 단위는 km이며 1,2,5,10의 단계가 존재함

type

캡쳐할 지도 타입. 위성 지도, 일반 지도, 위성 지도에 지형지물 정보가 포함되는 하이브리도 3가지가 존재함

 
GET /image/queue?companyType=kakao&lat=111.0&lng=11.0&layerMode=false&level=2&type=satellite_base HTTP/1.1
Host: localhost:8080

response

Path Type Description

[].uuid

String

제작된 위성 이미지의 uuid, 해당 값으로 자신이 제작 요청한 이미지를 발급받음

[].x

Number

분할 전송되는 이미지의 x 위치 값

[].y

Number

분할 전송되는 이미지의 y 위치 값

임시 저장

크롤링이 완료된 위성 이미지를 서버에 임시 보관합니다. 해당 기능은 이미지 생성 요청 시 같이 전달되는 토큰을 통해서만 접근할 수 있으며, 인증용 토큰이 헤더에 존재하지 않으면 요청은 거부됩니다.

header

Name Description

AUTH_TOKEN

서버 간 인증 토큰

request

Path Type Description

uuid

String

저장할 이미지의 uuid

base64EncodedImage

String

인코딩된 이미지

 
POST /image/storage HTTP/1.1
Content-Type: application/json;charset=UTF-8
AUTH_TOKEN: eyJhbGciOiJIUzUxMiJ9.eyJleHAiOjE3MDk2NDU1NDB9.hc8H3Z2eIWEQ16OLIHZLQj4bj3CH08i83RHDvBPqFDsGdqBE-n8U_Lknou98rQavuCYKsqYcNLXDD36_B-Te0w
Content-Length: 109
Host: localhost:8080

{
  "uuid" : "151daf08-ac9a-41a9-b11f-5fdfc1fafa8f",
  "base64EncodedImage" : "SSBhbSBWaXJ0dWFsIEltYWdl"
}

response

HTTP/1.1 200 OK
Vary: Origin
Vary: Access-Control-Request-Method
Vary: Access-Control-Request-Headers

발급

생성된 위성 이미지를 발급받습니다. 생성 요청 완료 후 전달 받았던 uuid 값을 통해 이미지를 발급하며, 유저가 이미지를 받음과 동시에 해당 이미지는 서버에서 삭제됩니다.

request

Table 1. /image/storage/{uuid}
Parameter Description

uuid

발급받을 이미지의 uuid

 
GET /image/storage/1f564fa7-39cc-47f2-8202-a15ba153698d HTTP/1.1
Host: localhost:8080

response

HTTP/1.1 200 OK
Vary: Origin
Vary: Access-Control-Request-Method
Vary: Access-Control-Request-Headers
Content-Type: image/jpeg
Content-Length: 18

I am Virtual Image

공지사항

목록 조회

공지사항 목록을 조회합니다. 세부적인 공지사항의 내용은 전달되지 않습니다. 전달받은 게시글 번호 미만으로 최대 10개의 목록을 반환하며, 게시글 번호를 0번으로 요청 시 가장 최신 게시글 10개의 목록이 반환됩니다.

request

Table 2. /notice/list/{startPostNumber}

GET /notice/list/11 HTTP/1.1
